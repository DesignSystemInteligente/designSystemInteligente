Os protótipos deste design system possuem uma conexão com os sistemas que são desenvolvidos a partir dele.

E a inteligência que os interligam se dá num processo de visão computacional que retorna para esse design system um relatório de inconsistências caso as encontre.

Dessa forma a quarta heurística de Nielsen é garantida automaticamente, reportando para o time de design a criação de tarefas de dívidas técnicas quando não há consistências e padrões.
